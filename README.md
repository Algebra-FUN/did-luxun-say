# “鲁迅说过吗？” Chrome Extension
### 鲁迅先生Avatar镇楼
![lx](./images/lx-128.png)
## 用途
用于快速搜素页面上的内容，快速检验页面上疑似“鲁迅的话”是否是真实的
## 安装
下载后，在Chrome拓展工具页面开启**开发者模式**，加载已解压的扩展程序，即可
